# Cannes Fixtures

Seed fixtures for the local Cannes MVP demo:

- `idea.seed.json`: funded idea input used for the demo flow
- `worker.seed.json`: seeded worker identity used for claim and submission
- `submission.valid.json`: valid scaffold submission used for deterministic scoring
- `submission.invalid.json`: malformed submission for rejection/rework checks
- `escrow.expected.json`: expected funded/reserved/released settlement totals
